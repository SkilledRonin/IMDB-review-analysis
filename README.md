# IMDB-review-analysis
Sentiment analysis on IMDB movie reviews using an LSTM deep learning model built with TensorFlow/Keras. Classifies reviews as positive or negative using text preprocessing, embeddings, and sequence modeling. Includes visualizations and a prediction function.

### Dataset
This project uses the [IMDB Movie Reviews Dataset](https://www.kaggle.com/code/priyang/data-analysis-imdb-movie-dataset).
Download it and place it in the project folder before running the notebook.
